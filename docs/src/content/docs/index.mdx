# UniPay SDK

UniPay SDK is a universal payment gateway SDK for Node.js that provides a unified interface for integrating multiple payment providers into your application. Currently supporting Midtrans with plans to add DOKU and Xendit providers.

## Quick Start

### Installation

```bash
npm install @fiandev/unipay-sdk
```

### Basic Usage

```javascript
import { Unipay, MidtransProvider } from '@fiandev/unipay-sdk';

// Initialize with Midtrans provider
const unipay = new Unipay({
  provider: 'midtrans',
  config: {
    client_id: 'your-client-id',
    secret_key: 'your-secret-key',
    is_production: false
  }
});

// Create a payment transaction
const payment = await unipay.createPayment({
  orderId: 'ORDER-123',
  amount: 10000,
  customerEmail: 'customer@example.com',
  customerName: 'John Doe',
  description: 'Test payment'
});

console.log(payment);
```
